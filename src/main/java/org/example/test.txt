процедура вещественный арифметикаИБулевы(вещественный x, вещественный y):
    вещественный z = 0.0
    если (x > y && !(x == y)) тогда:
        z = ((x + y) * (x - y)) / y
    иначе:
        z = -1.0
    вернуть z
